# SIMPLE-WEDDING-PLANNER-WEBSITE
A simple and elegant wedding planner website built using HTML, CSS, and JavaScript. Includes login validation, portfolio gallery, testimonials with images, and an inquiry form with real-time input validation. Fully responsive and easy to navigate.
 1. Login System (Front-End Validation)
Email format validation
Password length check
Redirects to the Home page after successful login
 2. Homepage
Beautiful hero section with background image
“Why Choose Us” service cards
Links to detailed pages (Elegant Planning, Creative Decor, Professional Team)
 3. Portfolio Page
Wedding gallery with multiple images
Responsive grid layout
 4. Testimonials Page
Client reviews
Images included under each testimonial
 5. Inquiry Form
Name, email, phone, and message validation
Alerts for incorrect inputs
Clean and user-friendly UI
 6. Modern UI/UX
Clean design
Soft color palette
Card-based layout
Responsive for mobile screens
